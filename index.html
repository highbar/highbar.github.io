highbar

Lodash and Underscore, ported to .NET and the JVM

View on GitHub Download .zip Download .tar.gz

The Goal

This project attempts to take the wonderful JavaScript utility library Underscore and its performance-minded cousin Lodash, identify the bits that are "missing" in .NET and the JVM, and then create them using the most modern techniques available on both of those platforms.
